# GRAC 등급분류 — 말달리자 (앱인토스 전용)

## 어떤 경로?

| 경로 | 출시 URL | 추천 |
|------|----------|------|
| A. GRAC 직접 신청 | 토스 미니앱 URL | 장르 선택 막히면 비추 |
| **B. Play 내부 테스트 + IARC** | Play 스토어 URL | **✅ 현재 선택** — `docs/play-internal-test.md` |

말달리자는 **B번(Play 내부 테스트 → GRAC 조회 → 앱인토스 콘솔)** 으로 진행합니다.

| 항목 | 값 |

## 「출시 URL」에 뭘 넣나?

GRAC 폼의 URL/유통 주소는 **공개 앱스토어 링크가 아니라, 토스에서 플레이하는 주소**를 쓰면 됩니다.

### 1단계: `.ait`를 콘솔에 먼저 올리기

```bash
npm run build
# dist/ 또는 horserace.ait → 앱인토스 콘솔 업로드 → QR 테스트
```

### 2단계: URL 선택

| 단계 | URL | 용도 |
|------|-----|------|
| 심사·GRAC 신청 중 | `https://horserace.private-apps.tossmini.com` | QR 테스트(비공개) |
| 정식 출시 후 | `https://horserace.apps.tossmini.com` | 운영 |

**GRAC 신청 시**에는 보통 **private-apps URL** + QR 테스트 방법을 담당자 메모/첨부 영상에 함께 적습니다.  
(토스 앱 → 샌드박스/QR로 접속 필요)

### GRAC가 URL 외에 요구하는 것

- **실행 가능한 게임물** (빌드 파일 또는 WebView 번들)
- **플레이 영상** (등급에 영향 주는 화면만, 로그인→경주→예상→친구방 등)
- **게임물내용설명서**
- **게임 주요화면 2종 × 2칸** → `assets/apps-in-toss/grac-screenshots/` (아래 표)
- 사업자: 게임물제작업자등록증 등

| 콘솔 칸 | 파일 |
|---------|------|
| 주요화면1 · 자체등급분류 | `01-race-self-rated.png` |
| 주요화면1 · 앱인토ss | `01-race-apps-in-toss.png` |
| 주요화면2 · 자체등급분류 | `02-tips-self-rated.png` |

GRAC 직접 신청(기타-앱인토ss)이면 **양쪽 칸 모두 같은 플레이 화면**이면 됩니다.

앱인토스 콘솔 3단계에는 GRAC에서 받은 **등급분류증명서 PDF**를 첨부합니다 (스토어 URL 경로 아님).

## 앱인토스 콘솔 등급 화면

2025년 12월부터 **등급 정보는 콘솔 입력 시 토스가 브릿지 화면으로 자동 표시**합니다.  
별도 splash 개발 불필요 — 콘솔에 등급분류번호·이용등급·내용정보만 정확히 입력.

## 말달리자 심사 시 강조 문구

- 베팅·마권·배당·현금 환전 **없음**
- 「예상」은 게임 내 점수·칭호·골드만
- 친구 방은 **같은 경주 맞추기**(앱이 돈을 중개하지 않음)
- 광고: 선택적 리워드(티켓·찌라시 P)

피할 표현: 내기게임, 배당, odds, 마권, 도박
