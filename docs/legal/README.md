# 약관 URL (앱인토스 콘솔 등록용)

GitHub Pages 활성화 후 아래 URL을 콘솔 **약관/동의문 → 서비스 이용약관**에 등록하세요.

| 약관 | URL |
|------|-----|
| 서비스 이용약관 | `https://aundots.github.io/horserace/legal/terms.html` |
| 개인정보처리방침 | `https://aundots.github.io/horserace/legal/privacy.html` |

## GitHub Pages 켜는 방법

1. GitHub → `aundots/horserace` → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / Folder: **/docs**
4. Save → 1~2분 후 위 URL 접속 확인

## 운영자 정보

사업자 정보는 `OPERATOR-INFO.md`에 정리되어 있으며, 약관·개인정보처리방침(MD/HTML)에 반영되어 있습니다.

## 콘솔 입력

- **서비스 이용약관**: 필수 → 위 terms.html URL
- **개인정보 수집·이용 동의**: 권장 → privacy.html URL (또는 콘솔 기본+링크)

토스 로그인 필수 약관(제3자 제공 등)은 콘솔에서 자동 포함됩니다.
